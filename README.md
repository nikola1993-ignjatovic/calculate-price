# Calculate Price

## Overview
A .NET C# application designed to calculate prices based on various parameters and rules. It features modular services, DTOs, and flexible configuration via `appsettings.json`.

## Features
- Modular price calculation system
- Uses DTOs for structured input/output
- Custom pricing rules via enums and helpers

## Project Structure
- `Dtos/`: Data Transfer Objects
- `Enums/`: Pricing constants
- `Helpers/`: Utility methods
- `Interfaces/`: Service definitions
- `Services/`: Pricing logic implementation
